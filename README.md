## Setup

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Build all packages:
   ```bash
   pnpm build
   ```

3. Configure environment variables:
   - Copy `.env.example` to `.env` in each app directory
   - Set required values (DISCORD_TOKEN, etc.)
